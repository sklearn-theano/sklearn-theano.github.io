sklearn_theano.datasets.fetch_asirra(image_count=1000)[source]¶
Parameters:

image_count : positive integer
Returns:

data : Bunch

Dictionary-like object with the following attributes : ‘images’, the sample images, ‘data’, the flattened images, ‘target’, the label for the image (0 for cat, 1 for dog), and ‘DESCR’ the full description of the dataset.
